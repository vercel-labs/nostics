## `Diagnostic`

Imported from `nostics`. Extends `Error`.

```ts
class Diagnostic extends Error {
  name: string
  message: string
  get why(): string
  fix?: string
  docs?: string
  sources?: string[]
  cause?: unknown

  constructor(init: DiagnosticInit, captureFrom?: Function)

  toJSON(): object
}
```

You don't usually construct a `Diagnostic` yourself; `defineDiagnostics` does that. You will use the type for `instanceof` checks and to type-narrow caught errors.

### Members

| Member    | Type        | Description                                                                                                          |
| --------- | ----------- | -------------------------------------------------------------------------------------------------------------------- |
| `name`    | `string`    | The code (e.g. `'MATH_E001'`). Defaults to `'Diagnostic'` when constructed directly.                                  |
| `message` | `string`    | The already-interpolated `why`. Inherited from `Error`.                                                              |
| `why`    | `string` (getter) | Alias for `message`.                                                                                          |
| `fix`     | `string \| undefined` | Resolved from the definition.                                                                              |
| `docs`    | `string \| undefined` | Resolved from `docsBase` or the per-code override.                                                         |
| `sources` | `string[] \| undefined` | Call-site `sources` array.                                                                               |
| `cause`   | `unknown`   | Call-site `cause`. Inherited via `Error`'s cause-aware constructor.                                                  |
| `stack`   | `string \| undefined` | Captured at construction. The top frame is the user code that invoked the handle, not nostics internals. |

### `toJSON(): object`

Returns a plain object suitable for serialization. `JSON.stringify(diagnostic)` calls this automatically.

```ts
{
  name: string
  why: string
  fix: string | undefined
  docs: string | undefined
  sources: string[] | undefined
  cause: unknown
  stack: string | undefined
}
```

Used by:

- `jsonFormatter` (which is `JSON.stringify` under the hood).
- `devReporter` (sends this object over `import.meta.hot.send`).
- `createFetchReporter` (POSTs this as the body).
- `createFileReporter` (writes this as one NDJSON line per diagnostic).

### Constructor

```ts
new Diagnostic(init: DiagnosticInit, captureFrom?: Function)
```

```ts
interface DiagnosticInit extends DiagnosticCallParams {
  why: string
  fix?: string
  docs?: string
}
```

`captureFrom` is the V8 stack-cutoff frame. Defaults to `Diagnostic` itself, so the top of the trace is the `new Diagnostic(...)` call site. `defineDiagnostics` passes its internal handle method to strip nostics' own frames as well. Ignored on engines without `Error.captureStackTrace`.

You will typically not call the constructor directly. Use `defineDiagnostics`.

## `DiagnosticInit`

The structured initializer for the constructor.

| Field     | Type        | Required | Description                                                  |
| --------- | ----------- | -------- | ------------------------------------------------------------ |
| `why`     | `string`    | yes      | Becomes `Error.message`.                                     |
| `fix`     | `string`    | no       | Actionable instructions.                                     |
| `docs`    | `string`    | no       | Docs URL.                                                    |
| `cause`   | `unknown`   | no       | Original error or context.                                   |
| `sources` | `string[]`  | no       | `file:line:column` locations.                                |

## Usage

Narrow a caught value:

```ts
import { Diagnostic } from 'nostics'

try {
  divide(1, 0)
}
catch (err) {
  if (err instanceof Diagnostic) {
    // typed access:
    console.log(err.name, err.fix, err.docs)
  }
}
```

Dispatch on the code:

```ts
catch (err) {
  if (err instanceof Diagnostic) {
    switch (err.name) {
      case 'MATH_E001':
        // ...
      case 'MATH_W001':
        // ...
    }
  }
}
```

Serialize for transport:

```ts
const payload = JSON.stringify(diagnostic) // calls toJSON()
ws.send(payload)
```
